¿Qué es Python?

Python es un lenguaje de programación fácil de aprender y muy utilizado.

¿Para qué sirve?

  • Crear programas
  • Automatizar tareas
  • Desarrollar inteligencia artificial
  • Analizar datos

1. Mostrar texto

print("Hola mundo")

Explicación: Muestra texto.

2. Variables

edad = 15

Explicación: Guarda información.

3. Condicionales

if edad >= 18: print("Mayor de edad")

Explicación: Ejecuta código según una condición.

4. Ciclos

for i in range(5): print(i)

Explicación: Repite instrucciones.
