¿Qué es HTML?

HTML es el lenguaje que se utiliza para crear páginas web. Es como el esqueleto de una página, ya que organiza todo el contenido que ves en internet.

Cuando entras a una página, el navegador lee el código HTML y lo convierte en lo que ves en pantalla.

¿Para qué sirve?

  • Mostrar textos
  • Insertar imágenes
  • Crear enlaces
  • Organizar contenido

1. Estructura básica

Definición: Es la base de toda página web.

Explicación: Toda página necesita una estructura para funcionar correctamente.

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> Hola mundo </body> </html>

¿Cómo funciona?
El navegador lee este código y muestra el contenido dentro del <body>.

2. Títulos y texto

Sirven para mostrar información al usuario.

<h1>Título principal</h1> <p>Este es un párrafo</p>

Explicación: h1 es el título más importante y p es para párrafos.

3. Imágenes

<img src="imagen.jpg" alt="Imagen">

Explicación: "src" indica la imagen que se mostrará.

4. Enlaces

<a href="https://google.com">Ir a Google</a>

Explicación: Permite ir a otra página web.

5. Listas

<ul> <li>Elemento 1</li> <li>Elemento 2</li> </ul>

Explicación: Sirve para organizar información en forma de lista.

¿Cómo empezar a aprender HTML?

  1. Crear un archivo .html
  2. Escribir código básico
  3. Abrirlo en el navegador
  4. Practicar todos los días
