¿Qué es CSS?

CSS es el lenguaje que se usa para dar estilo a una página web. Si HTML es la estructura, CSS es la apariencia.

¿Para qué sirve?

  • Cambiar colores
  • Modificar tamaños
  • Diseñar la página
  • Hacerla más atractiva

1. Cambiar color

p { color: red; }

Explicación: Cambia el color del texto.

2. Fondo

body { background-color: lightblue; }

Explicación: Cambia el fondo de la página.

3. Tamaño de texto

p { font-size: 20px; }

Explicación: Ajusta el tamaño del texto.

4. Bordes

div { border: 2px solid black; }

Explicación: Agrega bordes a los elementos.
