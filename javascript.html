¿Qué es JavaScript?

Es un lenguaje que permite agregar interactividad a una página web.

¿Para qué sirve?

  • Crear botones interactivos
  • Mostrar mensajes
  • Hacer juegos
  • Validar formularios

1. Variables

let edad = 15;

Explicación: Guarda datos.

2. Mensajes

alert("Hola mundo");

Explicación: Muestra un mensaje.

3. Condicionales

if (edad >= 18) { alert("Mayor de edad"); }

Explicación: Ejecuta código según una condición.

4. Ciclos

for (let i = 0; i < 5; i++) { console.log(i); }

Explicación: Repite acciones.

¿Cómo empezar?

  1. Aprender variables
  2. Usar alert()
  3. Practicar condiciones
  4. Crear botones
